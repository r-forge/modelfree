[b,lims] = binom_lims(r,m,x,gl,link,p,K,initval )

Maximum likelihood estimates of the parameters of the psychometric function with guessing and lapsing rates or only guessing rate. The estimated parameters for the linear part are in vector b and the estimated limits are in lims.

Input:

r: number of successes at points x

m: number of trials at points x

x: stimulus levels

Optional input:

gl: indicator, calulate only guessing if 'guess'and both guessing and lapsing otherwise

link: link function

p: degree of the polynomial to be fitted on the linear scale; default is p = 1

K: Power parameter in Weibull and reverse Weibull models; default is K = 2

initval: initial value for limits; default is [.01 .99] if guessing and lapsing rates are calculated and .01 if only guessing rate is calculated

Output:

b: vector of estimated coefficients for the linear part

lims: estimated guessing rate and 1âˆ’lapsing rate
