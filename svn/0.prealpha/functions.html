Model-free estimation of a psychometric function

modelfree (alpha)
Home Downloads Demonstration Documentation Examples Functions Contacts

Functions

bandwidth_bootstrap

Finds a bootstrap estimate of the optimal bandwidth h for a local polynomial estimate of the psychometric function with specified guessing and lapsing rates.

bandwidth_cross_validation

Finds the cross-validation bandwidth for a local polynomial estimate of the psychometric function with specified guessing and lapsing rates.

bandwidth_optimal

Finds the optimal bandwidth for a local polynomial estimate of the psychometric function with specified guessing and lapsing rates . The difference between this function and bandwidth_cross_validation is that here the true psychometric function is known.

bandwidth_plugin

Calculates an estimate of the AMISE optimal bandwidth for a local polynomial estimate of the psychometric function.

binom_lims

Maximum likelihood estimates of the parameters of the psychometric function with guessing and lapsing rates or only guessing rate. The estimated parameters for the linear part are in vector b and the estimated limits are in lims.

binom_revweib

Maximum likelihood estimates of the parameters of the reverse Weibull model for the psychometric function. The estimated parameters for the linear part are in vector b and the estimated exponent is K.

binom_weib

Maximum likelihood estimates of the parameters of the Weibull model for the psychometric function. The estimated parameters for the linear part are in vector b and the estimated exponent is K.

binomfit_lims

Fits a binomial generalised linear model with lower and upper constraints as in lims.

binomval_lims

Fitted values at points xfit for a binomial generalised linear model with coefficients b and lower and upper constraints as in lims.

bootstrap_ci_sl

Finds a bootstrap estimate of a confidence interval at a significance level alpha for the estimated slope for the local polynomial estimate of the psychometric function with guessing and lapsing rates specified in lims. Confidence interval is based on bootstrap percentiles.

bootstrap_ci_th

Finds a bootstrap estimate of a confidence interval at a significance level alpha for the estimated threshold for the local polynomial estimate of the psychometric function with guessing and lapsing rates specified in lims. The confidence interval is based on bootstrap percentiles.

bootstrap_sd_sl

Finds a bootstrap estimate of the standard deviation of the estimated slope for a local polynomial estimate of the psychometric function with guessing and lapsing rates specified in lims.

bootstrap_sd_th

Finds a bootstrap estimate of the standard deviation of the estimated threshold for the local polynomial estimate of the psychometric function with guessing and lapsing rates specified in lims.

comploglog_link

Complementary loglog link in cell form for use with GLMFIT, GLMVAL, and other Matlab GLM functions. The guessing and lapsing rates are fixed to values given in lims; hence link is a function of only one variable.

deviance

Calculates the deviance for fitted values of the psychometric function pfit.

locglmfit

Local polynomial estimator for the psychometric function and eta function (the psychometric function transformed by link) for binomial data; also returns the hat matrix H.

logit_link

Logit link in cell form for use with GLMFIT, GLMVAL, and other Matlab GLM functions. The guessing and lapsing rates are fixed to values given in lims; hence link is a function of only one variable.

loglog_link

Loglog link in cell form for use with GLMFIT, GLMVAL, and other Matlab GLM functions. The guessing and lapsing rates are fixed to values given in lims; hence link is a function of only one variable.

probit_link

Probit link in cell form for use with GLMFIT, GLMVAL, and other Matlab GLM functions. The guessing and lapsing rates are fixed to values given in lims; hence link is a function of only one variable.

revweibull_link

Reverse Weibull link in cell form for use with GLMFIT, GLMVAL, and other Matlab GLM functions. The guessing and 1− lapsing rates are fixed at values given in lims, and the exponent is set to be equal to K; hence link is a function of only one variable.

threshold_slope

Finds the approximate value of x (=x_th) for which the value of the psychometric function is equal to thresh and the approximate value of slope in x_th.

weibull_link

Weibull link in cell form for use with GLMFIT, GLMVAL, and other Matlab GLM functions. The guessing and 1− lapsing rates are fixed at values given in lims, and the exponent is set to be equal to K; hence link is a function of only one variable.
